Admonitor is an Add-on for Firefox browsers that has been specially written for the Values & Value project. It works alongisde the Values & Value Facebook App and gathers data that we are not able to collect directly from Facebook, this includes advertising shown on Facebook, the order of posts as they appear at the top of users' Facebook timelines, and advertising included on other sites via systems such as GoogleAds. We are gathering this data in order to relate patterns of user interaction with Facebook to possible patterns that might appear in the advertising users see as the browse the web, so that we can learn more about how Facebook seeks to stimulate user interaction in order to increase the economic value that can be generated from this.

The Admonitor Add-on is easy to install and requires no input or actions from the user after it has been registered. Instructions for installing and registering the software are given below along with more information on Data Collection and Privacy, as well as how to disable or remove the software and handling problems you might encounter when installing or using it.

Data Collection and Privacy

The software collects advertising data from Facebook that appears as Sponsored Stories in your main activity stream, the adverts that appear in the Facebook sidebar and ambient advertising that is pushed to you as you browse the web. We are gathering these to determine if there are any patterns relating user activity on Facebook to advertising content and if common patterns in advertising can be identified amongst users who are friends.

The software also collects the order of posts as they appear in your Facebook activity stream. We are gathering these to determine of there are patterns in how Facebook selects the posts that it displays to you and, as with adverts, if common patterns can be identified amongst users who are friends.

The detailed content of this data is only available to users registered with the Values & Value website. Users only have access to their own data and are not able to see others.

The ambient web advertising data includes the URLS of the adverts and the domain name of the site that it was displayed on, but does not collect the specific URL of the page itself. For example, if the software detects an advert from GoogleAds on a page with a URL such as http://www.example.com/pages/tolookat.html it will only store the domain name for the page: http://www.example.com.

The researchers are able to see the data for all participants but only in an anonymized form. Users's real names are replaced with random fictitious names and no content from Facebook posts, such as messages, or photos, are shown to researchers. The researchers are therefore able to see the patterns of interaction between users but without knowing who is who or what the content of this is.

The images and content from sponsored stories and advertising on Facebook is visible to researchers as this does not contain any personal information.

When viewing the web advertising data the researchers are able to see the names of the advertising companies but not the actual websites visited by participants.

It may be that during the study the researchers need to see some of the content from your Facebook activity, for example to discuss a specific posting or series of events that are relevant to the study. In this case the researchers will either meet with you in person, if doing an interview, or request that you turn off the anonymization for your data. This is controlled by a setting in your personal profile which only you have access to.

If you wish to temporarily prevent the software from gathering data, you can do so either by opening a Private Window from the main menu in Firefox. The software will not collect any data from pages you view in a Private Window. Alternativle, you can open the Add-ons dialogue and disable the plug-in (see below). Please remember to re-enable to software when you have finished.

Unregistering

Click on the Values & Value icon, or go to the Tools menu in your browser and select Values and Value to open the login screen (as described above), and click on the unregister button.

The software can now be removed or registered for a different user.

Disabling and Uninstalling

The software can be disabled or removed by opening the Add-ons dialogue box and clicking on either the Disable or Remove button.

In the most recent versions of Firefox (version 29.0 or above) the Adds-on dialogue box is found in the main menu in the top-right of the browser window. In older versions of Firefox (version 28.0 and below) this is found in the Tools menu.

Troubleshooting

The software won't install on my browser

The software will ony work with Firefox from versions 21.0 upwards and will not work with other browsers, such as Chrome, Safari or Internet Explorer. We recommend that you upgrade to the latest version of Firefox when using this software: https://www.mozilla.org/en-GB/firefox/new/.

The software started to install but then stopped or failed before completing

When you first click on the Install software button, a dialogue box will appear saying "Firefox prevented this site (values.doc.gold.ac.uk) from asking you to install software on your computer" with a button given you the options "Allow" and "Not now", click the Allow option:

You will then see a dialogue box saying "Install add-ons only from authors you trust" with the opetions to "Cancel" or "Install Now", click on Install Now:

The software should then download and install onto your browser. You can now follow the setup instructions above.

If you continue to have problems please contact us.

I get error messages appearing from the software

Please contact us and include a note of the error message, and the username you are registered with.

I wish to withdraw from the project and have my data removed

Please contact us and let us know the username you are registered with.
